# 灵猴机器人AI工作流平台 - 飞书分享指南

## 🚀 快速分享步骤

### 当前服务状态
- ✅ 本地服务器已启动
- 📍 端口：8080
- 🔗 本地访问：http://localhost:8080/index.html

### 飞书分享方案

#### 方案一：局域网分享
1. **获取本机IP地址**：
   - Windows: 打开命令提示符，输入 `ipconfig`
   - Mac: 系统偏好设置 → 网络，查看IP地址
   - 或者在终端输入：`ifconfig | grep inet`

2. **生成分享链接**：
   - 格式：`http://[您的IP地址]:8080/index.html`
   - 示例：`http://192.168.1.100:8080/index.html`

3. **在飞书中分享**：
   - 复制链接到飞书群聊或文档中
   - 同事点击即可访问完整的交互式界面

#### 方案二：使用ngrok内网穿透（公网访问）
1. **安装ngrok**：
   ```bash
   # 下载并安装ngrok
   # 访问 https://ngrok.com/ 注册账号
   ```

2. **启动内网穿透**：
   ```bash
   ngrok http 8080
   ```

3. **获取公网链接**：
   - ngrok会生成类似：`https://abc123.ngrok.io`
   - 完整链接：`https://abc123.ngrok.io/index.html`

#### 方案三：部署到云服务器
1. 将整个项目上传到云服务器
2. 配置Nginx或Apache
3. 绑定域名，生成永久访问链接

### 🎯 推荐使用方案一
- ✅ 最简单快捷
- ✅ 保持完整交互功能
- ✅ 适合内部团队分享
- ✅ 无需额外配置

### 📋 分享时注意事项
1. **确保服务器保持运行**：不要关闭终端窗口
2. **网络连接**：确保分享对象在同一局域网内
3. **防火墙设置**：可能需要允许8080端口访问
4. **浏览器兼容性**：建议使用Chrome、Safari、Edge等现代浏览器

### 🔧 故障排除
- **无法访问**：检查IP地址和端口是否正确
- **页面不显示**：确认index.html文件路径正确
- **样式丢失**：检查styles.css和app.js文件是否存在
